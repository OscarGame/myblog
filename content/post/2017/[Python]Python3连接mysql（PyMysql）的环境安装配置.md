安装Python3之后，下载PyMysql：  
https://github.com/PyMySQL/PyMySQL/

解压进入根目录，执行命令：

    python setup.py install
    
然后运行PyMysql安装目录下的example.py测试MySQL数据库链接是否成功（记得修改example.py里面的mysql密码）
    
    python example.py
